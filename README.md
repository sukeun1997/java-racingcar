```aidl
// 협력 역할 책임 행위 상태 클래스


협력 
----
게임담당 -> 입력뷰 ( 자동차 입력 )
    입력뷰 -> 검증 ( 구분자 (,) 검증 요청 )
    입력뷰 -> 검증 ( 자동차 이름 검증 요청 )
    입력뷰 -> 게임 담당 ( , 로 분리된 자동차 배열 응답 ) 
게임담당 -> 자동차 팩토리 ( 입력된 값으로 자동차를 생성 )
    자동차 팩토리 -> 자동차 ( 이름과 위치 0 을 가지는 자동차 생성)
    자동차 팩토리 -> 경주담당 ( 생성된 자동차들을 가지는 일급 컬렉션 생성 )
게임담당 -> 입력뷰 ( 시도할 횟수 입력 )
    입력뷰 -> 검증 ( 1~Max_value 사이인지 )
게임담당 -> 경주담당 ( 횟수 만큼 게임 진행 )
    경주담당 -> 자동차 ( 움직여라 )
    자동차 -> 랜덤수 ( 0~9 숫자 요청 )
    랜덤수 -> 자동차 ( 0~9 응답 )
    자동차 -> 경주담당 ( 움직인 후 응답 )
경주담당 -> 결과뷰 ( 자동차 이름 및 위치 출력 )
n 만큼 반복...
경주담당 -> 결과뷰 ( 우승자 발표 ) 


역할
--- 
자동차 팩토리 , 자동차, 결과뷰 , 입력뷰 , 검증 , 랜덤수 , 게임담당 , 경주담당 

책임
----

게임담당
    입력뷰 생성
    자동차 팩터리 생성
    입력뷰에게 메세지 송신 ( 자동차 이름 입력 , 게임횟수 요청 )
    자동차 팩터리에게 메세지 송신 ( 자동차 이름으로 자동차 생성을 요청 )
    경주담당에게 메시지 송신 ( 횟수에 따른 게임 진행 요청 )
    게임종료
    
자동차 팩토리
    자동차 이름에 따른 자동차들 생성 담당
    경주담당(일급 컬렉션) 생성 담당 
    
경주담당
    출력뷰 생성
    자동차들에게 움직이라고 명령
    출력뷰 호출 담당

자동차
    이름을 가진 자동차를 생성
    현재 이름과 위치 정보 제공 
    랜덤수에따른 움직임 
    
검증
    , 에따라 구분되는지 검증
    자동차 5글자 검증
    숫자 1~max_value 검증

입력뷰
    입력 스캐너 생성
    자동차 이름을 입력받음
    횟수를 입력받음
    
출력뷰
    실행 결과 출력 담당 
    우승자 출력 담당
    
랜덤수
    0~9 사이 랜덤수
    

    



```
